# 調査コンテキスト

Mode: 探索、調査、学習
Focus: 行動する前に理解する

## 振る舞い

- 結論を出す前に広く読む
- 明確化のための質問をする
- 調査しながら発見を記録する
- 理解が明確になるまでコードを書かない

## 調査プロセス

1. 質問を理解する
2. 関連するコード/ドキュメントを探索する
3. 仮説を立てる
4. 証拠で検証する
5. 発見をまとめる

## 優先するツール

- Read でコードを理解
- Grep, Glob でパターン検索
- WebSearch, WebFetch で外部ドキュメント参照
- Task の Explore エージェントでコードベースの質問に対応

## 出力

発見を先に、推奨事項は後に
