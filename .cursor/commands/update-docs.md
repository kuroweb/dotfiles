# ドキュメントの更新

信頼できるソースからドキュメントを同期:

1. package.jsonのscriptsセクションを読み込み
   - スクリプト参照テーブルを生成
   - コメントからの説明を含める

2. .env.exampleを読み込み
   - すべての環境変数を抽出
   - 目的と形式をドキュメント化

3. docs/CONTRIB.mdを以下の内容で生成:
   - 開発ワークフロー
   - 利用可能なスクリプト
   - 環境セットアップ
   - テスト手順

4. docs/RUNBOOK.mdを以下の内容で生成:
   - デプロイ手順
   - 監視とアラート
   - よくある問題と修正
   - ロールバック手順

5. 古くなったドキュメントを特定:
   - 90日以上変更されていないドキュメントを検出
   - 手動レビュー用にリスト化

6. 差分サマリーを表示

信頼できる唯一のソース: package.jsonと.env.example
